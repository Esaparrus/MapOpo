# MapOpo

Aplicación web interactiva para practicar geografía de Navarra a través de un mapa sin etiquetas. Al iniciarse, la app plantea preguntas sobre montes, ríos, carreteras o pueblos navarros. El usuario debe marcar en el mapa dónde cree que se encuentra el elemento, comprobar su respuesta y ver la distancia al objetivo.

## Ejecutar en local

```bash
python -m http.server --directory public 8000
```

Abre <http://localhost:8000> en tu navegador para jugar.
